Key Features

Powerful tools for bioinformatics graph exploration

Graph Visualization

Interactive graph visualization with multiple layout algorithms including dagre, klay, tidytree, and more. Explore your data dynamically.

JSON Editor

Built-in JSON editor for direct data manipulation. Modify graph structure on the fly and see changes reflected in real-time.

Resizable Panels

Adjust the size of graph, walks, and info panels to fit your workflow. Fully customizable workspace for efficient analysis.

Gene Annotation

Exon-level gene annotation with GENCODE data. Precise overlap detection using efficient binary search and two-pointer algorithms.
